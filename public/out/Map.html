new Map(canvasContainer, props, trackFPSCB) → {Map}

Main class for the whole engine, which initializes the whole structure and plugins You use the class by instantiating it with new and then initialize with init-method: var map = new Map(canvasElement, mapOptions ); promises = map.init( gameData.pluginsToActivate, mapData.startPoint ); Plugins can be added with activatePlugins-method by prodiving init(map) method in the plugin. Plugins are always functions, not objects that are instantiated. Plugins are supposed to extend the map object or anything in it via it's public methods.
