Show:

UI Class

Defined in: src\components\map\core\UI.js:12

Main class for showing UI on the map. Like unit selections and such. Has nothing to do with showing off-map data. Good examples for what this shows are: selected units-list, selection highlight (like a circle on the selected unit) and bringing the unit on top in the map. UI themes must implement this core UI module
